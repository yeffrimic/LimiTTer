# LimiTTer
Transmisor para el sistema Freestyle Libre FGM

Con LimiTTer puedes transformar el sistema Freestyle Libre FGM en un verdadero
Sistema de monitorización continua de glucosa (CGMS). Eso significa que no tiene que hacerlo manualmente, escanee el sensor con el lector. LimiTTer hace esto por usted, cada 5 minutos. Envía
el valor de glucosa a su teléfono inteligente que debe tener instalada la aplicación xDrip. xDrip es realmente
una aplicación excelente y madura para Android. Puede enviar los valores de glucosa a su reloj inteligente o
a través de Nightscout en la nube. La característica más importante es tener una guardia nocturna.
xDrip + es una versión mejorada de xDrip y ahora la aplicación predeterminada para LimiTTer.
Lo encontrará aquí: </br> </br>
<a href="https://jamorham.github.io/#xdrip-plus"> https://jamorham.github.io/#xdrip-plus </a> </br> <br>
Haga clic en "Descargar el APK más reciente" o elija "Nightly Snapshots" para ver las más recientes.

Tenga en cuenta que Abbott NO mantiene LimiTTer. Es un proyecto de DIY experimental.
Construirá su propio LimiTTer individual. Entonces es responsable de lo que
han construido. He abierto este proyecto aquí porque es una invitación para todos los desarrolladores.
por ahí para mejorar el LimiTTer.
